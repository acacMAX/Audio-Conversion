# 🎵 音频转换工具

一个功能强大的通用音频格式转换器，支持所有主流音频格式之间的互相转换。

## ✨ 功能特点

- **多格式支持**：支持 MP3, FLAC, WAV, OGG, M4A, AAC, WMA, OPUS, AIFF 等格式互相转换
- **拖拽上传**：支持直接拖拽文件或文件夹到程序窗口
- **批量转换**：一次性转换多个文件
- **自定义质量**：根据格式提供不同的质量选项（比特率、采样率）
- **智能输出**：所有转换后的文件统一保存到"音乐"文件夹
- **实时进度**：显示转换进度和详细日志
- **深色主题**：现代化界面设计

## 📦 安装依赖

```bash
pip install tkinterdnd2
```

## 🚀 使用方法

1. 运行程序：
```bash
python audio_converter.py
```

2. 选择文件：
   - 点击"📄 选择文件"按钮选择单个或多个文件
   - 点击"📂 选择文件夹"按钮选择整个文件夹
   - 或直接拖拽文件/文件夹到窗口

3. 设置转换选项：
   - 选择输出格式
   - 设置质量级别
   - 可选：勾选"转换完成后删除原文件"

4. 开始转换：
   - 点击"▶️ 开始转换"按钮
   - 等待转换完成
   - 转换后的文件保存在"音乐"文件夹中

## 🎯 支持的格式

### 输入格式
- MP3, FLAC, WAV, OGG, M4A, AAC, WMA, OPUS, AIFF, AU

### 输出格式
- **MP3**: libmp3lame 编码器，支持 64/128/192 kbps
- **FLAC**: 无损压缩，支持原始音质/44.1kHz/48kHz
- **WAV**: PCM 编码，支持原始音质/44.1kHz/48kHz
- **OGG**: Vorbis 编码器，支持 128/192/256 kbps
- **M4A**: AAC 编码器，支持 128/192/256 kbps
- **AAC**: AAC 编码器，支持 64/128/192 kbps
- **WMA**: Windows Media Audio，支持 64/128/192 kbps

## 🔧 系统要求

- Python 3.7+
- FFmpeg（用于音频转换）
- Windows 10/11 或 macOS 或 Linux

## 📝 注意事项

- 需要安装 FFmpeg 并确保在系统 PATH 中
- 转换大文件可能需要较长时间
- 建议在转换前备份重要文件
